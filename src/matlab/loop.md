# Loops / Conditionals
----------------------


### `for`
```matlab
for i=start:counter:finish
	statement1
	statement2
	...
end
```

### `while`
```matlab
while condition
	statement1
	statement2
	...
end
```

### `if else`
```matlab
if condition1
	statement1
elseif condition2
	statement2
else
	statement3
end
```
